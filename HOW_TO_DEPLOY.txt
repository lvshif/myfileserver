📦 WHAT YOU HAVE
----------------
A single self-contained installer file: fileserver-installer.sh (73KB)

This file contains EVERYTHING needed to deploy the file server to any Linux
machine with Python 3.6+. No additional files required!


🚀 QUICKEST WAY TO DEPLOY
--------------------------

Step 1: Copy the installer to target server
   scp fileserver-installer.sh user@target-server:/tmp/

Step 2: SSH to target server and run
   ssh user@target-server
   sudo bash /tmp/fileserver-installer.sh

Step 3: Access the file server
   http://target-server-ip/

That's it! 🎉


📋 DETAILED DEPLOYMENT OPTIONS
-------------------------------

Option 1: Default Installation
   sudo bash fileserver-installer.sh
   
   Installs to: /opt/fileserver
   Port: 80
   Base directory: /data


Option 2: Custom Installation
   sudo bash fileserver-installer.sh /custom/path 8080 /storage
   
   Installs to: /custom/path
   Port: 8080
   Base directory: /storage


Option 3: No Root Required (Port >= 1024)
   bash fileserver-installer.sh /home/user/fileserver 8080 /home/user/files
   
   No sudo needed when using ports >= 1024


Option 4: Remote Installation via curl
   # First, host fileserver-installer.sh on a web server
   # Then on target server:
   curl -sSL http://your-server/fileserver-installer.sh | sudo bash


🎮 MANAGING THE SERVER
----------------------

After installation, use these commands:

Start:    /opt/fileserver/start.sh
Stop:     /opt/fileserver/stop.sh
Restart:  /opt/fileserver/restart.sh
Status:   /opt/fileserver/status.sh

Or if systemd service was installed:

Start:    sudo systemctl start fileserver
Stop:     sudo systemctl stop fileserver
Restart:  sudo systemctl restart fileserver
Status:   sudo systemctl status fileserver


🌍 DEPLOYING TO MULTIPLE SERVERS
---------------------------------

Example: Deploy to 3 servers

#!/bin/bash
servers="server1.example.com server2.example.com server3.example.com"

for server in $servers; do
    echo "Deploying to $server..."
    scp fileserver-installer.sh user@$server:/tmp/
    ssh user@$server "sudo bash /tmp/fileserver-installer.sh"
    echo "✅ $server deployed"
done

🗑️ UNINSTALLATION
------------------

1. Stop the server:
   /opt/fileserver/stop.sh
   
   Or: sudo systemctl stop fileserver

2. Remove files:
   sudo rm -rf /opt/fileserver
   sudo rm /etc/systemd/system/fileserver.service
   sudo systemctl daemon-reload

3. (Optional) Remove data directory:
   sudo rm -rf /data


🆘 TROUBLESHOOTING
------------------

Problem: Python not found
Solution: sudo apt install python3  (Ubuntu/Debian)
          sudo yum install python3  (CentOS/RHEL)

Problem: Port 80 already in use
Solution: sudo lsof -i :80  (find what's using it)
          Or use different port: bash fileserver-installer.sh /opt/fileserver 8080

Problem: Permission denied
Solution: Use sudo for ports < 1024
          Or use port >= 1024 without sudo

Problem: Can't access from remote
Solution: Check firewall: sudo ufw allow 80/tcp
          Or: sudo firewall-cmd --permanent --add-port=80/tcp
