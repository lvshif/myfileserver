   fileserver-installer.sh (73KB)

🚀 DEPLOY IN 3 STEPS
--------------------

1️⃣  Copy to target server:
   scp fileserver-installer.sh user@target-server:/tmp/

2️⃣  SSH and run:
   ssh user@target-server
   sudo bash /tmp/fileserver-installer.sh

3️⃣  Access in browser:
   http://target-server-ip/

DONE! 🎊

🎯 QUICK EXAMPLES
-----------------

Example 1: Default installation
   sudo bash fileserver-installer.sh

Example 2: Custom port (no sudo needed)
   bash fileserver-installer.sh /home/user/fileserver 8080

Example 3: Deploy to multiple servers
   for server in server1 server2 server3; do
       scp fileserver-installer.sh user@$server:/tmp/
       ssh user@$server "sudo bash /tmp/fileserver-installer.sh"
   done

🔧 AFTER INSTALLATION
---------------------

Manage the server:
   /opt/fileserver/start.sh      Start server
   /opt/fileserver/stop.sh       Stop server
   /opt/fileserver/restart.sh    Restart server
   /opt/fileserver/status.sh     Check status

Or use systemd:
   sudo systemctl start fileserver
   sudo systemctl stop fileserver
   sudo systemctl status fileserver
